# detection-engine (Go) — legacy

Этот каталог — **устаревшая** реализация детектора и коррелятора на Go. Активный путь в репозитории и в CI: **`detection-engine-rs`** (Rust), образ `deploy/docker/Dockerfile.detection`.

Сохранено для справки и возможного сравнения поведения; новые правила и изменения вносите в `detection-engine-rs` и при необходимости обновляйте YAML в `sigma-rules/`.
